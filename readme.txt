A WordPress plugin for Dhepa Latest Newspaper Theme. 
  
== Description ==
  
This simple plugin is required for Dhepa latest newspaper theme users. 
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
  
== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
You have to buy Dhepa Latest Newspaper Theme from https://dhepa.com
  
= How to uninstall the plugin? =
  
This plugin is required for Dhepa Latest newspaper theme. but you can simply deactivate and delete the plugin.
